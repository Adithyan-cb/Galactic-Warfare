# Galactic Warfare - Asteroid Dodger

Welcome to Galactic Warfare, a thrilling space combat game where you pilot a spaceship and navigate through an asteroid field. Your objective is to dodge the incoming asteroids for as long as possible while earning a high score. How long can you survive?

## Installation

To play Galactic Warfare, you need to have the following requirements:

- Python (latest version)
- Pygame
- JSON
- Math
- Random

Please ensure that you have the latest version of Python installed on your system. You can download Python from the official website: [python.org](https://www.python.org/downloads/)

Once Python is installed, you can use the following command to install the necessary packages:

```bash
pip install pygame json math random
```

## Controls

Use the following controls to navigate your spaceship:

- **Left Arrow Key**: Move the spaceship to the left.
- **Right Arrow Key**: Move the spaceship to the right.

## Gameplay

As soon as you start the game,a spaceship appears. Your spaceship will be positioned at the center of the screen, and asteroids will begin approaching from the top.

Your objective is to maneuver your spaceship and avoid colliding with the incoming asteroids. Use the left and right arrow keys to move your spaceship horizontally across the screen. Be quick and agile to dodge the asteroids effectively.

## Scoring

Your score in Galactic Warfare is based on how long you can survive without colliding with an asteroid. The longer you avoid collisions, the higher your score will be. The score is measured in seconds, and it will be displayed on the screen during gameplay.

Challenge yourself to beat your own high score and compete with your friends to see who can survive the longest!
